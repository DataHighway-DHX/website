DATA HIGHWAY
#채굴

토큰 채굴은 전체 블록체인 이코시스템에 가치를 더하는 중요한 참가 방법입니다. Data Highway 공인 IoT 토큰 또는 다른 부가 가치 토큰을 락업하는 것은 참가의 한 방법이며, 토큰 채굴이라 부릅니다. Data Highway 에 밸리데이터 또는 노미네이터로 스테이킹을 할 때, 토큰 채굴자는 더 많은량의 DHX 토큰을 받을 수 있는 스피드 부스트를 받게됩니다. 자격 조건은 초기에는 사전 설정된 기준을 기반으로 하지만, 추후에는 DAO 에 의해 락업 기간, 락업 토큰 수량, 참여 수준, 그리고 프로모션이 정해집니다.

Data Highway 의 공인 부가 가치 토큰은 다음과 같습니다:
MXC, IoTA, Polkadot, Waltonchain, Streamr

Data Highway
#로밍

Data Highway 의 사물인터넷 파라체인은 LoRaWAN 로밍을 위한 분산화된 DNS 를 겸비한 분산 로밍 허브를 제공하여, LoRaWAN 네트워크 운영자가 Data Highway 블록체인의 안전한 분산 데이터베이스에 로밍 계약을 저장 및 검색할 수 있게 하며 계약, 청구, 분쟁 해결 및 조정과 관련 로그를 관리할 수 있게 합니다.


데이터 제공업체에 속한 LoRaWAN 사물인터넷 기기는 Data Highway 에 일정한 로밍 계약을 체결한 네트워크 운영자를 통해 활성화 되어, 구독 서비스를 구매한 구독자에게 지속적인 데이터 전송을 위해 업링크를 유지하면서 로밍 서비스를 구축합니다. Data Highway 의 인터체인 데이터 마켓은 다른 블록체인에 기반을 둔 디앱 (DApp) 을 운영하는데에도 큰 도움을 줍니다.
